# 2. IELTS考试
## 2.1. 考试的内容
一共四个方面，分为笔试和口语。笔试部分一共2个半小时：

- 听力。多听
- 阅读。3篇文章，一共60min。所以一篇文章要10min看完，10min做题目。
	- 考察的是语法和单词，这两个不熟，那么是不可能在那么短时间内看完文章。就算你大致看懂了，也达到不了解题的标准。当然，做题目也是有技巧的。
- 写作。2篇作文。小作文和大作文。单词数量要求不一样
	- 要自己掌握语法和单词

口语，需要提前3天预约。一共15分钟：

- 口语：
	- 自我介绍
	- 卡片上的topic，可以准备1min
	- 自由交谈
