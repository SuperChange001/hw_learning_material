# 2. 使用方法

1.  听：先听一遍单词，把听不出的记下来。打X
2.  背：然后对着单词书，把听不出来，但是认识的打O，这个说明要加强记忆
    1.  看英文解释，注意词的type和tense。
    2.  看一遍例句，完整读一遍，背一遍。遇到好的例句，抄录下来
    3.  做每个单元的练习
3.  写：听着单词，在电脑里打出来。看看拼写是否正确
4.  听：最后巩固的听力
